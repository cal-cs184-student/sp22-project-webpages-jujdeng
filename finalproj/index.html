CS 184: Computer Graphics and Imaging, Spring 2022

Final Project: Non-Photorealistic Rendering

Julia Deng, Harmony He, Jessica Liang, Christina Shao

Summary

Our project will be on NPR, non-photorealistic rendering. Rather than rendering images and scenes to reflect things realistically like a photograph, we are exploring the artistic side of graphics.



Problem Description

The problem we want to solve is rendering images and scenes in different artistic styles that are not necessarily realistic. We all appreciate art and enjoy graphic design. We’ve used effects in Adobe Illustrator that can be applied to images, prompting our idea for this project. We think it’s important to express creativity and explore different styles, so this project can be a way to portray objects in more abstract ways such that they are still recognizable.

This problem is challenging because we entirely cannot rely on real-world behavior. We’ll solve this by defining our own behavior for what we want to render, such as the color and thickness of pencil scribbles, what colors to use to posterize a photo, and how much the watercolor should bleed.



Goals and Deliverables

We plan on accomplishing at least the posterization and pencil effects. Depending on the results of our research, the input files will either be 3D .dae files or 2D .jpg/.png images. The output will most likely be an image file.

Posterization is where we represent a subject in only several colors instead of the full spectrum, with clear divisions between each color region. In Adobe Illustrator, this is equivalent to image tracing with a small number of colors. One of the main tasks to implement posterization is picking the main colors used. We need to select colors that are similar to those used most in the actual scene but are also different enough from each other so that we can show contrast. Once the small set of colors are selected, we will create an algorithm to assign a color for each point of the scene. A primitive approach could be to assign the color closest to the original value, but this could result in uneven boundaries, meaning we would need to implement some sort of edge detection to achieve smooth edges between color regions.

The Pencil Effect makes our subject appear as if it were a sketched, pencil drawing. In order to implement the pencil effect, we must implement techniques for silhouette edge detection and interior shading.

The quality of our results will be measured based on how visually similar the output simulates the planned effect. Since the goal is non-photorealistic rendering, this metric is fairly lenient and emphasizes aesthetics over accuracy.

We hope to also accomplish the watercolor effect. This will render a scene to look like watercolor artwork where the color bleeds from the outer edge inwards. The larger parts of the scene should be white to simulate paper, and we can select the main colors similar to posterization to apply the blending to.
